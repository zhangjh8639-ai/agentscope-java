# Ecosystem

This group covers extensions that don't belong to a specific category like memory or RAG, but extend the runtime, language ecosystem, debugging tools, or training pipeline:

| Extension | Purpose |
| --- | --- |
| [Chat Completions Web](chat-completions-web.md) | Expose your Agent through an OpenAI Chat Completions–compatible API |
| [Kotlin](kotlin.md) | Provide `suspend` / `Flow`-style extension functions for Kotlin users |
| [Studio](studio.md) | Hook your Agent into AgentScope Studio for visual debugging |
| [Training](training.md) | Sample production traffic and feed it into a training backend |
