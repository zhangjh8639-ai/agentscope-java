# 生态扩展

放在这一组里的扩展不属于"记忆/会话/RAG/协议/基础设施"任一具体大类，而是对运行环境、语言生态、调试工具、训练流水线的扩展：

| 扩展 | 用途 |
| --- | --- |
| [Chat Completions Web](chat-completions-web.md) | 把 Agent 暴露成 OpenAI Chat Completions 兼容接口 |
| [Kotlin](kotlin.md) | 给 Kotlin 用户提供 `suspend` / `Flow` 风格的扩展函数 |
| [Studio](studio.md) | 把 Agent 接入 AgentScope Studio，便于可视化调试 |
| [Training](training.md) | 把生产流量自动采样喂给训练后端，闭环优化 |
